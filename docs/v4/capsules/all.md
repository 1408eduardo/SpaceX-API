# Get all capsules

**URL** : `/capsules`

**Method** : `GET`

**Auth required** : NO

## Success Responses

**Condition** : User can not see any capsules.

**Code** : `200 OK`

**Content** : `[]`

### OR

**Condition** : User can see one or more capsules.

**Code** : `200 OK`

**Content** : All capsules are shown

```json
[]
```
